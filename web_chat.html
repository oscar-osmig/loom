Training File Format

JSON Format

An array of objects, each with subject, relation, and object:

[
  {"subject": "dogs", "relation": "is", "object": "mammals"},
  {"subject": "dogs", "relation": "has", "object": "fur"},
  {"subject": "dogs", "relation": "can", "object": "bark"},
  {"subject": "dogs", "relation": "color", "object": "brown"},
  {"subject": "dogs", "relation": "lives_in", "object": "houses"}
]
TXT Format

One fact per line, separated by | or ,

# Lines starting with # are ignored
dogs | is | mammals
dogs | has | fur
dogs | can | bark
dogs | color | brown
Common Relations
is has can color lives_in eats needs causes used_for has_property found_in makes
Tips
  • Use is for categories: "eagle is birds"
  • Use has for parts: "eagle has sharp talons"
  • Use can for abilities: "eagle can fly"
  • Use has_property for traits: "eagle has_property powerful"
  • Use color for colors: "eagle color brown"
  • Drag & drop files or use the Train button
  • You can also paste JSON directly in the chat
